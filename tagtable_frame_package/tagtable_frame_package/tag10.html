배경 이미지 작업하기
웹페이지 전체 또는 특정 영역에 배경 이미지와 배경색을 넣는 방법입니다.
형식 구성 내용
background <body background="bg.jpg"> 문서 전체 배경 이미지를 지정하는 과거 방식입니다.
bgcolor <body bgcolor="#eeeeee"> 문서 배경색을 지정합니다.
background-image style="background-image:url(bg.jpg)" CSS로 배경 이미지를 지정합니다.
background-repeat style="background-repeat:no-repeat" 배경 이미지 반복 여부를 지정합니다.
background-position style="background-position:center" 배경 이미지 위치를 지정합니다.
background-size style="background-size:cover" 배경 이미지 크기를 조절합니다.

예제 코드

<body background="bg.jpg">
  배경 이미지가 들어간 문서입니다.
</body>

<div style="background-image:url(bg.jpg); background-repeat:no-repeat; background-size:cover;">
  CSS 배경 이미지 영역
</div>
