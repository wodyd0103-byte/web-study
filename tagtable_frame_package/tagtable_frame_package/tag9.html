글자의 모양을 정의해 주는 태그
글자를 굵게, 기울임, 밑줄, 위첨자, 아래첨자 등으로 표현하는 태그입니다.
형식 구성 내용
<b> <b>굵은 글자</b> 글자를 굵게 표시합니다.
<i> <i>기울임 글자</i> 글자를 기울여 표시합니다.
<u> <u>밑줄 글자</u> 글자 아래에 밑줄을 표시합니다.
<sup> x<sup>2</sup> 위첨자를 표시합니다.
<sub> H<sub>2</sub>O 아래첨자를 표시합니다.
<s> / <strike> <s>취소선</s> 취소선을 표시합니다.
<font> <font face="궁서" color="red">글자</font> 글꼴, 색상, 크기를 지정하는 과거 방식입니다.

예제 코드

<b>굵은 글자</b><br>
<i>기울어진 글자</i><br>
<u>밑줄 글자</u><br>
수식 예시: x<sup>2</sup> + H<sub>2</sub>O<br>
