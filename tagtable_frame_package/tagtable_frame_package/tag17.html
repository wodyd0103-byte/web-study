입력 폼 만들기
사용자에게 값을 입력받기 위해 사용하는 입력 폼 관련 태그입니다.
형식 구성 내용
<form> <form action="save.html" method="post"> 입력 양식 전체를 묶습니다.
<input> <input type="text" name="user"> 한 줄 입력칸, 비밀번호, 체크박스, 라디오 버튼 등을 만듭니다.
type text / password / checkbox / radio / submit 입력칸의 종류를 지정합니다.
<textarea> <textarea rows="5" cols="30"></textarea> 여러 줄 입력칸을 만듭니다.
<select> <select><option>선택</option></select> 드롭다운 선택 상자를 만듭니다.
<option> <option value="html">HTML</option> 선택 상자의 항목을 만듭니다.
<button> <button>확인</button> 버튼을 만듭니다.
<label> <label for="id">이름</label> 입력칸의 설명 글자를 연결합니다.

예제 코드

<form action="result.html" method="post">
  <label>이름: <input type="text" name="username"></label><br>
  <label>비밀번호: <input type="password" name="pw"></label><br>
  관심 분야:
  <select name="subject">
    <option>HTML</option>
    <option>CSS</option>
  </select><br>
  <textarea rows="4" cols="30">의견을 입력하세요.</textarea><br>
  <input type="submit" value="전송">
</form>
