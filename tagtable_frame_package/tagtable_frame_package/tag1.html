문서의 기본 구조를 나타내 주는 태그
HTML 문서를 만들 때 가장 먼저 이해해야 하는 기본 구조 태그입니다. 문서 전체, 머리말 영역, 본문 영역, 제목 영역을 구분합니다.
형식 구성 내용
<!DOCTYPE html> <!DOCTYPE html> HTML 문서의 형식을 브라우저에게 알려 줍니다.
<html> <html> ... </html> HTML 문서 전체를 감싸는 최상위 태그입니다.
<head> <head> ... </head> 문서 정보, 제목, 문자 인코딩, 스타일 연결 등을 작성하는 부분입니다.
<title> <title>문서 제목</title> 브라우저 탭에 나타나는 제목을 지정합니다.
<meta> <meta charset="UTF-8"> 문자 인코딩, 설명, 키워드 등 문서 정보를 설정합니다.
<body> <body> ... </body> 화면에 실제로 보이는 내용을 작성하는 부분입니다.
<h1>~<h6> <h1>제목</h1> 제목을 표시합니다. h1이 가장 크고 h6이 가장 작습니다.
<!-- --> <!-- 주석내용 --> 브라우저에는 보이지 않는 설명을 작성할 때 사용합니다.

예제 코드

<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <title>나의 첫 HTML 문서</title>
</head>
<body>
  <h1>태그표 홈페이지</h1>
  <!-- 이 부분은 화면에 보이지 않는 주석입니다. -->
  <p>HTML 기본 구조를 연습합니다.</p>
</body>
</html>
참고: HTML 태그를 설명하려면 <, > 문자를 그대로 쓰면 태그로 인식될 수 있습니다. 화면에 문자로 보이게 하려면 &lt;, &gt;, &amp;, &quot;, &nbsp; 같은 특수문자 표기법을 사용합니다.
