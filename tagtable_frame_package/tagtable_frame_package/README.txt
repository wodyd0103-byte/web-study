태그표 홈페이지 만들기 패키지

[실행 방법]
1. 압축을 해제합니다.
2. tagtable.html 파일을 더블클릭해서 실행합니다.
3. 왼쪽 menu.html 메뉴를 클릭하면 오른쪽 main 프레임에 tag1.html ~ tag19.html 설명 페이지가 열립니다.

[구성 파일]
- tagtable.html : 처음 실행하는 프레임 정의 파일
- menu.html : 왼쪽 메뉴 파일
- tag1.html ~ tag19.html : 각 태그 설명 페이지
- common.css : 메뉴와 설명 페이지의 공통 디자인
- index.html : 편의용 실행 파일, 열면 tagtable.html로 이동

[작성 기준]
- 과제 설명에 맞춰 frameset/frame 구조를 사용했습니다.
- 각 tag 파일은 table 태그를 이용해 형식, 구성, 내용을 정리했습니다.
- 메뉴는 총 19개 항목으로 구성했습니다.
- HTML 태그가 화면에 문자로 보이도록 &lt;, &gt; 같은 특수문자를 사용했습니다.
