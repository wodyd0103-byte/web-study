문서를 가운데로 정렬시켜 주는 태그
문장, 이미지, 표 등을 가운데에 배치할 때 사용하는 태그와 속성입니다.
형식 구성 내용
<center> <center>내용</center> 내용을 가운데 정렬합니다. 예전 HTML 방식입니다.
align="center" <p align="center">내용</p> 문단이나 영역의 정렬을 가운데로 지정합니다.
<div align> <div align="center">내용</div> 여러 요소를 묶어서 가운데 정렬합니다.
CSS text-align <div style="text-align:center">내용</div> CSS를 이용한 권장 방식의 가운데 정렬입니다.

예제 코드

<center>
  <h2>가운데 제목</h2>
  <p>이 문장은 가운데에 표시됩니다.</p>
</center>

<div style="text-align:center">
  CSS 방식 가운데 정렬
</div>
