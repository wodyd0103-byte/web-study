표 만들기 태그
행과 열로 정보를 정리하는 표 관련 태그입니다. 태그 설명 페이지를 만들 때 가장 많이 활용할 수 있습니다.
형식 구성 내용
<table> <table> ... </table> 표 전체를 만드는 태그입니다.
<caption> <caption>표 제목</caption> 표의 제목을 표시합니다.
<tr> <tr> ... </tr> 표의 한 행을 만듭니다.
<th> <th>제목 셀</th> 굵게 표시되는 제목 셀을 만듭니다.
<td> <td>내용 셀</td> 일반 내용 셀을 만듭니다.
border <table border="1"> 표 테두리 두께를 지정합니다.
cellpadding <table cellpadding="8"> 셀 안쪽 여백을 지정합니다.
cellspacing <table cellspacing="0"> 셀과 셀 사이 간격을 지정합니다.
rowspan <td rowspan="2">내용</td> 세로 방향으로 셀을 합칩니다.
colspan <td colspan="3">내용</td> 가로 방향으로 셀을 합칩니다.

예제 코드

<table border="1" cellpadding="8" cellspacing="0">
  <caption>태그 설명표</caption>
  <tr>
    <th>태그</th>
    <th>설명</th>
  </tr>
  <tr>
    <td>&lt;table&gt;</td>
    <td>표를 만드는 태그</td>
  </tr>
</table>
