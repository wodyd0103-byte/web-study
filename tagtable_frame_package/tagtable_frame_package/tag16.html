문서를 임의로 고정시키는 태그
입력한 공백, 줄바꿈, 코드 형태를 일정하게 유지하여 보여 주는 태그입니다.
형식 구성 내용
<pre> <pre>내용</pre> 공백과 줄바꿈을 입력한 그대로 표시합니다.
<code> <code>코드</code> 컴퓨터 코드나 태그 예시를 표시합니다.
<kbd> <kbd>Ctrl + S</kbd> 키보드 입력을 나타냅니다.
<samp> <samp>출력 결과</samp> 프로그램 출력 결과를 나타냅니다.
<nobr> <nobr>줄바꿈 방지</nobr> 텍스트가 자동 줄바꿈되지 않게 합니다. 과거 방식입니다.
<wbr> 긴단어<wbr>분리 필요할 때 줄바꿈될 수 있는 위치를 지정합니다.

예제 코드

<pre>
이 안에서는
  공백과 줄바꿈이
    그대로 유지됩니다.
</pre>

<p>HTML 태그 예시: <code>&lt;p&gt;문단&lt;/p&gt;</code></p>
