이미지 맵을 처리하는 태그
하나의 이미지에서 특정 영역을 클릭할 수 있게 만드는 이미지 맵 관련 태그입니다.
형식 구성 내용
<img usemap> <img src="map.jpg" usemap="#menuMap"> 이미지와 이미지맵을 연결합니다.
<map> <map name="menuMap"> ... </map> 클릭 가능한 영역들의 묶음을 정의합니다.
<area> <area shape="rect" coords="0,0,100,80" href="a.html"> 이미지 안에서 클릭 영역 하나를 만듭니다.
shape rect / circle / poly 사각형, 원형, 다각형 영역을 지정합니다.
coords coords="x1,y1,x2,y2" 클릭 영역의 좌표를 지정합니다.

예제 코드

<img src="menu.jpg" usemap="#tagmap" alt="태그 메뉴">
<map name="tagmap">
  <area shape="rect" coords="0,0,120,50" href="tag1.html" alt="기본 구조">
  <area shape="circle" coords="200,80,40" href="tag2.html" alt="문단 태그">
</map>
