문단 혹은 줄을 바꾸는 태그
문단을 구분하거나 줄바꿈을 할 때 사용하는 태그입니다. 글 내용을 보기 좋게 나누는 데 필요합니다.
형식 구성 내용
<p> <p>문단 내용</p> 하나의 문단을 만듭니다. 문단 앞뒤로 기본 여백이 생깁니다.
<br> 첫 줄<br>둘째 줄 줄을 강제로 바꿉니다. 닫는 태그를 사용하지 않습니다.
<pre> <pre>내용</pre> 공백과 줄바꿈을 입력한 그대로 보여 줍니다.
<div> <div>영역</div> 블록 단위의 영역을 만들 때 사용합니다.
<span> <span>일부분</span> 문장 안의 일부 영역만 따로 지정할 때 사용합니다.

예제 코드

<p>HTML은 웹 문서를 작성하는 기본 언어입니다.</p>
<p>CSS는 HTML 문서의 모양을 꾸며 줍니다.</p>
