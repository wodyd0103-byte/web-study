목록을 정리해 주는 태그
여러 항목을 순서 있게 또는 순서 없이 정리할 때 사용하는 목록 태그입니다.
형식 구성 내용
<ul> <ul> ... </ul> 순서가 없는 목록을 만듭니다. 보통 점 모양 기호가 붙습니다.
<ol> <ol> ... </ol> 순서가 있는 목록을 만듭니다. 숫자나 알파벳이 붙습니다.
<li> <li>항목</li> 목록 안의 개별 항목을 작성합니다.
<dl> <dl> ... </dl> 용어 설명 목록을 만듭니다.
<dt> <dt>용어</dt> 설명 목록의 용어를 작성합니다.
<dd> <dd>설명</dd> 설명 목록의 자세한 내용을 작성합니다.

예제 코드

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

  <li>메뉴 선택</li>
  <li>오른쪽 프레임 확인</li>
</ol>
