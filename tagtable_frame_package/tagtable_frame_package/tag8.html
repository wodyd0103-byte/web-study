인용하기 태그
다른 사람의 말, 책 문장, 참고 자료를 인용할 때 사용하는 태그입니다.
형식 구성 내용
<blockquote> <blockquote>긴 인용문</blockquote> 긴 인용문을 별도의 문단처럼 표시합니다.
<q> <q>짧은 인용문</q> 짧은 인용문을 문장 안에 표시합니다.
<cite> <cite>출처</cite> 작품명이나 출처를 표시할 때 사용합니다.

예제 코드

<blockquote>
  HTML은 웹페이지의 구조를 만드는 언어입니다.
</blockquote>
<p>선생님은 <q>태그의 역할을 이해하는 것이 중요하다</q>고 말씀하셨습니다.</p>
<cite>HTML 기초 수업 자료</cite>
