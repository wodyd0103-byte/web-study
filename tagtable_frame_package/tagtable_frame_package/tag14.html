색상 지정하기
글자색, 배경색, 링크색 등을 지정하는 방법입니다.
형식 구성 내용
color <font color="red">글자</font> 글자 색상을 지정합니다.
bgcolor <body bgcolor="yellow"> 문서나 표의 배경색을 지정합니다.
text <body text="black"> 문서 전체 글자색을 지정합니다.
link / vlink / alink <body link="blue" vlink="purple"> 링크, 방문한 링크, 클릭 중인 링크 색을 지정합니다.
CSS color style="color:#ff0000" CSS 방식으로 글자색을 지정합니다.
CSS background-color style="background-color:#ffffcc" CSS 방식으로 배경색을 지정합니다.

예제 코드

<body bgcolor="#f5f5f5" text="black" link="blue">
  <font color="red">빨간색 글자</font><br>
  <span style="color:#0066cc; background-color:#ffffcc">CSS 색상 지정</span>
</body>
