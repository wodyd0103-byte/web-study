이미지 다루기에 필요한 태그
이미지를 삽입하고 크기, 테두리, 정렬, 설명 등을 조절하는 태그와 속성입니다.
형식 구성 내용
<img> <img src="photo.jpg" alt="사진"> 웹페이지에 이미지를 삽입합니다.
src src="image.jpg" 이미지 파일 경로를 지정합니다.
alt alt="이미지 설명" 이미지가 보이지 않을 때 대체 텍스트를 표시합니다.
width / height width="300" height="200" 이미지의 가로, 세로 크기를 지정합니다.
border border="1" 이미지 테두리를 지정합니다. 과거 방식입니다.
align align="left" 이미지의 정렬 위치를 지정합니다. 과거 방식입니다.
hspace / vspace hspace="10" vspace="10" 이미지 주변의 가로/세로 여백을 지정합니다.
<figure> <figure>이미지와 설명</figure> 이미지와 설명을 하나의 묶음으로 표현합니다.
<figcaption> <figcaption>이미지 설명</figcaption> figure 안에서 이미지 설명을 작성합니다.

예제 코드

<img src="sample.jpg" alt="샘플 이미지" width="300" height="200" border="1">

<figure>
  <img src="html.png" alt="HTML 로고" width="200">
  <figcaption>HTML을 설명하는 이미지</figcaption>
</figure>
