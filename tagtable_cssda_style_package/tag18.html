tag18.html

창 만들기 태그
하나의 브라우저 화면을 여러 영역으로 나누거나, 페이지 안에 다른 페이지를 넣는 태그입니다.
형식 구성 내용
<frameset> <frameset cols="280,*"> ... </frameset> 브라우저 화면을 여러 프레임으로 나눕니다. 이번 과제의 tagtable.html에서 사용합니다.
<frame> <frame src="menu.html" name="menu"> frameset 안에 들어갈 각각의 프레임을 지정합니다.
name name="main" 링크의 target과 연결할 프레임 이름을 정합니다.
target <a href="tag1.html" target="main"> 링크 결과를 어느 프레임에 보여 줄지 지정합니다.
<noframes> <noframes>안내문</noframes> 프레임을 지원하지 않는 브라우저를 위한 내용을 작성합니다.
<iframe> <iframe src="page.html"></iframe> 현재 문서 안에 다른 문서를 작은 창처럼 삽입합니다.

예제 코드

<!-- tagtable.html 예시 -->
<frameset cols="280,*">
  <frame src="menu.html" name="menu">
  <frame src="tag1.html" name="main">
</frameset>

<!-- menu.html 링크 예시 -->
<a href="tag2.html" target="main">문단 태그</a>
참고: frameset/frame 태그는 현재 HTML5에서는 권장되지 않지만, 이 과제 조건처럼 프레임 구조를 연습할 때 사용할 수 있습니다.
