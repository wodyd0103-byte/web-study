글자의 크기를 마음대로 조절하는 태그
제목 크기나 글자 크기를 조절할 때 사용하는 태그와 속성입니다.
형식 구성 내용
<h1>~<h6> <h1>가장 큰 제목</h1> 제목의 크기와 중요도를 나타냅니다.
<font size> <font size="5">글자</font> 글자의 크기를 지정합니다. 과거 방식이지만 기초 HTML 연습에서 자주 사용됩니다.
<big> <big>큰 글자</big> 기본 글자보다 조금 크게 표시합니다.
<small> <small>작은 글자</small> 기본 글자보다 작게 표시합니다.
style <span style="font-size:20px">글자</span> CSS를 사용하여 글자 크기를 더 자유롭게 지정합니다.

예제 코드

<h1>HTML 태그표</h1>
<h3>글자 크기 연습</h3>
<font size="5">font 태그로 크게 표시</font><br>
<span style="font-size:24px">CSS로 글자 크기 조절</span>
