선 그리기 태그
내용을 구분하기 위해 가로선을 넣는 태그입니다.
형식 구성 내용
<hr> <hr> 문서 안에 가로 구분선을 삽입합니다.
width <hr width="70%"> 선의 가로 길이를 지정합니다.
size <hr size="5"> 선의 두께를 지정합니다.
color <hr color="blue"> 선의 색상을 지정합니다.
align <hr align="left"> 선의 정렬 위치를 지정합니다.

예제 코드

<h2>첫 번째 내용</h2>
<p>위쪽 내용입니다.</p>
<hr width="80%" size="3" color="gray">
<h2>두 번째 내용</h2>
<p>아래쪽 내용입니다.</p>
