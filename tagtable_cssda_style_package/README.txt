태그표 홈페이지 - CSS Design Awards 스타일 개선 버전

실행 방법
1. 압축을 해제합니다.
2. tagtable.html 파일을 더블클릭해서 실행합니다.
3. 왼쪽 메뉴를 누르면 오른쪽 프레임에 tag1.html ~ tag19.html 또는 profile.html이 표시됩니다.

파일 구성
- tagtable.html : 프레임을 정의하는 시작 파일
- menu.html : 왼쪽 메뉴 페이지
- tag1.html ~ tag19.html : 각 태그 설명 페이지
- profile.html : 제작자 소개 페이지
- common.css : 전체 디자인 스타일
- images/profile.png : 제작자 프로필 사진

디자인 적용 내용
- CSS Design Awards에서 자주 보이는 다크 톤 기반의 고급스러운 레이아웃
- 큰 타이포그래피, 그라데이션 포인트 컬러, 글래스 카드 디자인
- 메뉴 hover 효과, 둥근 카드, 코드 박스 스타일 개선
- 과제 요구사항에 맞추어 frameset/frame 구조 유지
